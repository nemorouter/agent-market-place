Nemo Agent Widget — separately deployable embed

This page loads the widget exactly the way a customer would on their own site — one <script> tag, zero framework. The widget streams from the central Nemo MCP gateway (/v1/agents/{id}/respond); the gateway is not deployed here.

Click the Ask AI launcher in the bottom-right.

<script src="agent-widget.global.js"></script>
<script>
  NemoAgentWidget.mount({
    // public site: proxy keeps the sk-nemo key server-side
    proxyPath: '/api/public/ask',
    // OR direct: apiBase, agentId, apiKey: 'sk-nemo-…'
    title: 'Ask AI about Acme',
    suggestions: ['How does pricing work?', 'Is it OpenAI-compatible?'],
  });
</script>
