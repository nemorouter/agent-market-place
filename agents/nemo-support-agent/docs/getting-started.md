# Getting started with Nemo Router

> PLACEHOLDER docs — replace with real Nemo Router content, or set `WEBSITE_URL` to
> `https://nemorouter.ai/docs` and run `/api/ingest` to pull the live docs.

Nemo Router is a unified LLM gateway: one API key, credits, and dashboard across
many model providers.

## Create your first key
1. Sign in at https://nemorouter.ai and open **Keys**.
2. Click **Create virtual key** — it starts with `sk-nemo-`.
3. The full key is shown once at creation; copy it now.

## Make a request
Point any OpenAI-compatible client at `https://api.nemorouter.ai/v1` and use your
`sk-nemo-` key as the bearer token.

## Credits
Requests draw down credits on your key. Set a per-key budget to cap spend.
