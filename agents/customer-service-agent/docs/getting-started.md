# Getting started with Acme

Acme is a project-management tool for small teams. This page is sample content so
the agent has something to answer from out of the box — replace `./docs` with your
own Markdown, or point `WEBSITE_URL` at your live docs and run `/api/ingest`.

## Creating your first project

1. Sign in at https://acme.com and click **New Project**.
2. Give it a name and invite teammates by email.
3. Add tasks, assign owners, and set due dates.

## Inviting your team

Open **Settings → Members → Invite**. Invited users get an email; they join your
workspace once they accept. Roles are **Owner**, **Admin**, and **Member**.

## Keyboard shortcuts

- `c` — create a task
- `/` — search
- `g` then `p` — go to Projects
